Help me

If you want to work on the template, download the full mockup package!

Note: although the package includes individual images for the various icons, most of them are combined into the following 4 sprites:

menu/menu_sprite.png
Contains all the icons for the main menu, when they are not hovered.
menu/menu_sprite_hover.png
Contains all the icons for the main menu, when they are hovered.
content/content_sprite.png
Contains all heading icons (all the png's inside the "content" folder)
icons/icons_sprite.png
Contains all the general icons for action links (all the png's inside "icons" folder)

The sprites themselves are generated using the CSS Sprite Generator by Project Fondue, unchecking the "Wrap columns to fix Opera bug" option, and checking the "Compress Image with OptiPNG" one.
